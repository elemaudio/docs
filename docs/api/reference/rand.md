# el.rand([props])

:::tip Stable
Full support across available targets
:::

Generates a stream of random numbers uniformly distributed on the range [0, 1].

#### Props

| Name     | Default   | Type   | Description                            |
| -------- | --------- | ----------------------------------------------- |
| seed     | undefined | Number | Seeds the random number generator      |
